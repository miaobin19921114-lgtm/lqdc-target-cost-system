## 三、V1 产品定位

产品名称：地产目标成本测算系统

一句话定位：面向个人、小团队、小开发商、咨询顾问的轻量化地产目标成本测算工具。

核心价值：把复杂的地产目标成本 Excel 测算模板，变成一个可录入、可复算、可导入导出、可版本锁定、可交付成果的轻量系统。

V1 商用定位：V1 不是大型房企 ERP，也不是全周期成本管理平台。V1 是一个可以用于演示、试用、项目测算和轻量交付的目标成本测算工具。

## 五、V1 对外卖点与功能清单

V1 对外只讲四个能力：

1. 快速建立项目和测算版本
2. 完成收入、成本、税费、利润测算
3. 支持 Excel 导入、预览、确认落库和导出
4. 支持版本锁定和成果交付

对外宣传语：

地产目标成本测算系统，帮助小团队快速完成项目收入、成本、税费、利润测算，支持 Excel 导入导出和版本锁定，让复杂测算更清晰、更稳定、更容易交付。

## 十一、V1 功能模块规划

### 模块 1：登录与基础访问

包含：登录、登出、基础账号页、健康检查。

不包含：多角色权限、多组织管理、企业级权限矩阵、审批流。

### 模块 2：项目中心

包含：项目列表、新建项目、进入项目、项目基础信息展示、项目状态展示。

不包含：知识库、招采、合约、审批、AI、系统模板市场。

### 模块 3：版本管理

包含：版本列表、新建版本、复制版本、锁定版本、当前版本识别、锁定后只读提示。

不包含：高级版本差异分析、多版本图表对比、自动生成版本报告。

### 模块 4：项目概况

包含：项目基础信息、建筑面积指标、业态指标、车位指标、基础工程量指标、保存和校验。

V1 原则：只保留完成测算所需的基础字段，不继续无限增加明细字段。

不包含：楼栋全生命周期管理、地下室复杂分区、指标库大平台、CAD 自动识别、工程量自动提取。

### 模块 5：测算控制中心

包含：业态启用状态、成本分摊口径、条件性科目控制、税率口径提示、利润测算口径提示。

不包含：复杂规则治理平台、模板规则发布、多版本规则快照、自动规则校验中心。

### 模块 6：收入明细

包含：销售收入、商业收入、车位收入、其他收入、含税收入、不含税收入、销项税额、收入合计。

不包含：去化节奏、回款计划、销售现金流、动态销售计划。

### 模块 7：成本明细

包含：土地成本、前期成本、建安成本、安装成本、设备成本、精装修成本、室外管网、景观、道路、围墙大门、含税金额、不含税金额、税额、建面单方、可售单方。

不包含：合同管理、采购包管理、动态成本变更、付款计划、结算管理。

### 模块 8：成本分摊

包含：按建筑面积分摊、按可售面积分摊、按受益对象分摊、分摊前金额、分摊后金额、分摊结果校验。

不包含：复杂分摊规则治理、多套分摊方案对比、自动分摊策略推荐。

### 模块 9：税金测算

包含：增值税、附加税、土地增值税、所得税、税前利润、税后利润、销售净利率。

不包含：税务筹划方案、多口径税务模拟、税务报告自动生成、复杂清算模拟器。

### 模块 10：目标成本汇总

包含：销售收入、开发成本、税前利润、所得税、税后净利、销售净利率、建面单方成本、可售单方成本、一级 / 二级成本汇总。

不包含：高级老板看板、多项目看板、BI 报表、经营驾驶舱。

### 模块 11：业态利润分析

包含：业态收入、业态成本、业态税费、业态利润、业态利润率。

不包含：多项目业态横向对比、市场竞品分析、AI 业态优化建议。

### 模块 12：Excel 工作台

包含：模板下载、Excel 上传、导入预览、错误提示、确认导入落库、导出 Excel、导入批次、导入撤销。

不包含：AI 自动识别任意 Excel、多模板智能匹配、Word 报告导出、PPT 自动生成。

### 模块 13：成本科目及测算词典

包含：成本科目树、科目编码、科目名称、测算口径说明、规则说明。

不包含：复杂规则治理、规则发布、规则回滚、模板版本治理。

## 十七、V1.0.1 维护规则

V1 定稿后，不立即开发新功能。

只在以下情况进入 V1.0.1：

1. P0：系统打不开、登录失败、核心页面不可用。
2. P1：核心测算结果错误、Excel 导出错误、锁定版本防护失效。
3. P2：重要体验问题，确实影响试用。
4. P3：文案、样式、小交互优化，可以集中处理。

不进入 V1.0.1：

1. AI
2. 动态成本
3. 合约招采
4. 现金流
5. 审批流
6. CAD
7. 移动端
8. 多租户
9. 指标库大平台
10. 高级报表

## 十八、V1 商用试用判断

V1 允许进入商用试用，但不建议大规模公开推广。

适合：

1. 给熟人演示
2. 给小团队试用
3. 给 1 到 3 个真实项目试用
4. 用来验证付费意愿
5. 用来验证 Excel 模板引擎价值

不适合：

1. 大范围公开推广
2. 承诺大型企业级能力
3. 承诺 AI、合约、动态成本
4. 承诺替代完整 ERP

V1 试用目标：

1. 验证用户是否愿意使用系统替代 Excel
2. 验证用户是否愿意为测算模板付费
3. 验证最小测算主流程是否顺畅
4. 收集 V1.0.1 和 V2 的真实优先级
