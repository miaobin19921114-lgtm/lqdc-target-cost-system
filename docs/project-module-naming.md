# 项目模块命名标准

本文件用于统一系统页面标题、按钮文字和导航名称。

## 基础入口

- 项目测算中心
- 经营总控
- 项目中心

## 经营决策

- 投决评审
- 经营报告
- 敏感性分析

## 报告输出

- 打印经营报告
- 敏感性分析报告
- 税务报告

## 税费利润

- 税费测算总表
- 土地增值税清算测算表
- 业态利润分析

## 成本测算

- 目标成本编制
- 目标成本汇总
- 科目映射

## 项目基础

- 项目概况
- 业态维护 / 税务清算对象
- 版本管理
- 指标校验

## 导入与自检

- Excel 导入 / 导出
- 导入批次
- 汇总校验

## 执行要求

- 页面标题、按钮、返回入口、空状态、导出和打印入口必须使用以上标准名称。
- 新增页面时优先复用本文件的标准模块名称。
- 命名调整只改展示文案，不改变业务计算逻辑。
